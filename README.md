# PokeCable Room Repository

O projeto foi reorganizado dentro de [PokeCable/README.md](PokeCable/README.md).

- `PokeCable/api` — API e WebSocket do servidor

- `PokeCable/frontend` — frontend web
- `PokeCable/docs` — documentação
- `PokeCable/save` — saves e fixtures locais para teste
