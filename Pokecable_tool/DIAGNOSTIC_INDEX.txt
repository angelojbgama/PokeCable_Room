ROOT CAUSES OF FAILURES:

1. SaveModel.write_to_disk() API Mismatch [2 failures]
   - Applicator calls save.write_to_disk(path) but method signature is save.write_to_disk()
   - Affects: Crystal GS Ball, LeafGreen Mystic Ticket
   - Fix: Don't save in apply_event(); let caller handle it

2. Item Inventory Overflow [4 failures]
   - Test saves have full or near-full item bags
   - Cannot add event items (GS Ball, Eon Ticket, Old Sea Map, Member Card, Oak's Letter)
   - Affects: Sapphire, Emerald, Platinum
   - Fix: Create clean test saves with empty item bags

3. e-Reader Byte Offset Error [1 failure]
   - SaveModel.bytes buffer offset mismatch for e-Reader trainer injection
   - Affects: Sapphire e-Reader battle test
   - Fix: Debug offset calculation (0x3030 + slot*40) for SaveModel layout
